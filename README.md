## Project Setup

Clone the project from GitHub repository:

      git clone https://github.com/fajarutamaa/backend-challenge-6.git

Change Directory:

      cd backend-challenge-6

Install all package dependencies:

      npm install

Compile and hot-reload for development:

      npm run dev
